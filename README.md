# coDY - Architecture of Code

### [Võ Duy Bình (coDY) - Software Tech Student | Content Creator | AI Engineer]

A digital architecture portfolio redefining the boundaries between functional engineering and artistic expression. Rooted in the Bauhaus tradition where **"form follows function"**, this project is a structural showcase of modern web technologies, AI integration, and brutalist design.

---

## 🏛️ Design Philosophy: Bauhaus & Brutalism
This application is strictly built on Bauhaus principles:
- **Geometric Clarity**: Using a rigid grid system as the foundation.
- **Reduction to the Essence**: Removing decorative noise to focus on functional structure.
- **Primary Palette**: Focused on Red, Blue, and Yellow accents against a high-contrast Black and white backdrop.
- **Truth to Materials**: Code as the primary medium, expressed through logical layouts and raw typography.

---

## 🛠️ Technical Stack
- **Framework**: [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Brutalist pattern)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 6](https://reactrouter.com/)
- **Language**: TypeScript

## 🚀 Getting Started in AI Studio (Antigravity)

This project is optimized for development within **Google AI Studio (Build)**. 

### Instructions for AI Coding Agent:
If you are opening this project in AI Studio to continue development, you can instruct the agent as follows:

> "I have imported my portfolio. Please scan the `src/pages` and `src/constants.ts` to understand the current structure and coDY's personal data. Ensure all new features respect the Bauhaus design system defined in `tailwind.config.js` and `index.css`."

### Local Development:
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📁 Project Structure
- `/src/pages`: Individual structural modules (Home, Projects, Experience, About, Contact).
- `/src/components`: Global architectural elements (Navbar, Footer).
- `/src/constants.ts`: The central data registry for all personal information, projects, and skills.
- `/src/index.css`: Global design system and Bauhaus variables.
